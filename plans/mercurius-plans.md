# Mercurius Plans

Mercurius is a free, open-source project released under the MIT License. There are no commercial pricing tiers, subscription plans, or paid editions.

## Open Source (Free)

- **License:** MIT
- **Cost:** Free — no purchase, subscription, or API key required
- **Hosting:** Self-hosted on your own Node.js/Fastify infrastructure
- **Support:** Community support via GitHub Issues and Discussions

## What Is Included

All features are available in the single open-source release:

| Feature | Included |
|---|---|
| GraphQL query / mutation support | Yes |
| JIT compilation via graphql-jit | Yes |
| Query parsing and validation cache | Yes |
| Automatic dataloader / N+1 prevention | Yes |
| Subscriptions (WebSocket) | Yes |
| Federation (gateway + services) | Yes |
| Batched queries | Yes |
| Persisted queries | Yes |
| Authentication plugin (@mercuriusjs/auth) | Yes |
| Validation plugin (@mercuriusjs/validation) | Yes |
| Caching plugin (@mercuriusjs/cache) | Yes |
| TypeScript support & codegen | Yes |
| GraphiQL / Playground UI | Yes |
